# Tic-Tac-Toe 🎮

A Tic-Tac-Toe game built with **Next.js** and **TypeScript**.


## Features

- Classic 3x3 Tic-Tac-Toe gameplay
- Responsive design for mobile and desktop

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** CSS

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/asmaaalbermawy/tic_tac_toes.git
cd tic_tac_toes
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to play.
